Landing page moderna y elegante para **Pastelería Lilliam**, diseñada para destacar productos de repostería artesanal mediante una interfaz fluida, visualmente atractiva y optimizada para la conversión de clientes.

## Características

- **Diseño Responsive:** Adaptado perfectamente para móviles, tablets y escritorio.
- **Galería de Productos:** Exhibición visual de pasteles y postres con alta calidad.
- **Performance:** Desarrollado con **Vite** para una carga instantánea y una experiencia de usuario suave.
- **UI/UX Refinada:** Paleta de colores y tipografía seleccionadas para transmitir la esencia artesanal de la marca.

## Stack Tecnológico

- **Frontend:** [React.js](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Despliegue:** [Vercel](https://vercel.com/)
